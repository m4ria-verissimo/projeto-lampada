# Projeto Lâmpada 💡

Simulador interativo de uma lâmpada que liga, desliga e quebra com JavaScript puro. Projeto desenvolvido para praticar manipulação do DOM e eventos.

### 🔗 Acesse o projeto
[Ver online](https://m4ria-verissimo.github.io/projeto-lampada/)

### 🚀 Funcionalidades
- Liga e desliga a lâmpada ao clicar nos botões
- Troca de imagem ao passar o mouse por cima
- Lâmpada "quebra" se clicar duas vezes quando ligada
- Botão de reset para restaurar a lâmpada
- Interface responsiva

### 🛠️ Tecnologias utilizadas
- HTML5
- CSS3 
- JavaScript

### 📚 O que aprendi com este projeto
- Manipulação de eventos: `click`, `mouseover`, `mouseleave`, `dblclick`
- Troca dinâmica de atributos `src` de imagens com JavaScript
- Controle de estado da aplicação com variáveis
- Estrutura condicional `if/else` para lógica da lâmpada quebrada
- Organização de funções para cada ação
